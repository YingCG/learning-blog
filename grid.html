Setting UP Flex Grid

Flexbox vs Grid: Which should you use?

display: flex;

to put thing in a row(next to each other).
Flexbox — One Dimensional. That means content that goes ONE direction — horizontal or vertical. An example of best use for flexbox is when creating a Navbar .

display: grid;

to organizing page containers. Grid Layouts — Two Dimensional. to create a layout with both Rows and Columns, use the Grid system.

Connect stylesheet to HTML page

  1. Create your folder → "Third Website"
  2. In vscode → open folder
  3. Create a new file → index.html
  4. Type "!" to get your html boilder plate
  5. Connect your css file at the head of your html. For example:
    <!DOCTYPE html>
    <html>
    <head>
    <link rel="stylesheet" href="mystyle.css">
    </head>
    <body>

    <h1>This is a heading</h1>
    <p>This is a paragraph.</p>

    </body>
    </html>
