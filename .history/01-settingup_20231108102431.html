Create Blog in GitHub

Setup Repo

  1. Go to GitHub, create an account, Create a new GitHub pages repository:
  2. Use the syntax your-github-username.github.io for your repository name
  3. Create a new folder in your machine
  4. Open folder in Visual Studio Code
  5. Add a file call: index.html 
    <html>
      <head>
        <title>My blog</title>
      <body>
        <h1>Kia ora Taiao! (Hello World!)</h1> 
        <p> This is the beginning of my website home page.</p>
        <p> It will eventually contain a list of links to all my blogs. </p>
      </body>
    </html>

Commit and Push to GitHub

  1. Commit and Push the changes to GitHub
  2. Navigate to your [USERNAME].github.io repo on GitHub
  3. Enter the settings menu for that repo and click on the Pages tab in the left sidebar
  4. Under the Source option, select the main branch
  5. You can check out your live repo by visiting http://[USERNAME].github.io*

Link HTML and CSS

  1. Attach your main.css to the top of the HTML 
    ...
    <link href="your-stylesheet-link-here.css" rel="stylesheet" type="text/css">
    ...
  2. If stuck, This video demo's linking CSS to HTML at 5.08
